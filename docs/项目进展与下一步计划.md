### 具体步骤

#### 步骤1: 创建数据导入脚本 (优先级: 🔴 高)
**任务**: 创建将清洗后的数据导入MySQL的Python脚本

**文件路径**: `scripts/a03_export_to_mysql.py`

**核心功能**:
1. 数据库连接管理（SQLAlchemy）
2. 分批读取CSV文件（chunksize）
3. 数据预处理（类型转换、NULL处理）
4. 分批导入数据库（批量插入）
5. 进度显示和日志记录
6. 数据验证（导入前后对比）

**关键技术点**:
- 使用 `sqlalchemy.create_engine` 创建连接
- 使用 `pd.read_csv(chunksize=50000)` 分批读取
- 使用 `df.to_sql(method='multi', chunksize=1000)` 批量插入
- 连接池管理（pool_size, pool_pre_ping）
- 事务控制（自动提交/回滚）

**参考**: 数据导入模块面试要点.md 中的实现细节
